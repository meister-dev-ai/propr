Review focus: Android WebView JavaScript settings

What to look for:
- Enabling JavaScript execution in a WebView can result in cross-site scripting attacks.
- Enabling JavaScript in an Android WebView allows the execution of JavaScript code in the context of the running application. This creates a cross-site scripting vulnerability. For example, if your application's WebView allows for visiting web pages that you do not trust, it is possible for an attacker to lead the user to a page which loads malicious JavaScript. You can enable or disable Javascript execution using the`setJavaScriptEnabled`method of the settings of a WebView.
