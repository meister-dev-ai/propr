Review focus: EJB sets socket factory or URL stream handler factory

What to look for:
- An EJB should not set the socket factory used by ServerSocket or Socket, or the stream handler factory used by URL.
- The Enterprise JavaBeans 3.0 core specification, Section 21.1.2, states: The enterprise bean must not attempt to set the socket factory used by ServerSocket, Socket, or the stream handler factory used by URL. These networking functions are reserved for the EJB container. Allowing the enterprise bean to use these functions could compromise security and decrease the container's ability to properly manage the runtime environment.
