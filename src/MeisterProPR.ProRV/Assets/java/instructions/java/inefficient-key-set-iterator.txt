Review focus: Inefficient use of key set iterator

What to look for:
- Iterating through the values of a map using the key set is inefficient.
- Java's Collections Framework provides several different ways of iterating the contents of a map. You can retrieve the set of keys, the collection of values, or the set of "entries" (which are, in effect, key/value pairs). The choice of iterator can affect performance. For example, it is considered bad practice to iterate the key set of a map if the body of the loop performs a map lookup on each retrieved key anyway.
